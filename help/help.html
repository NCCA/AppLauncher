Using Apps'Ere!

Overview

Apps'Ere! is a simple application launcher designed to help you quickly find and start your favorite programs.

Main Menu

  • Debug → Show Output: Toggle the debug output panel.
  • Help → About: View information about the app and its author.
  • Help → Using the App: Open this help page.
  • Theme: Switch between System, Light, and Dark themes.

Launching Applications

Browse or search for applications in the main window. Click an app to launch it instantly.

Right clicking on any of the apps will give you the option to add to the favourites. You can also add a desktop icon. Not that this will need to be authorized on first use by right clicking on the desktop icon and "allow launch"

Installer scripts may launch a popup on use with further instructions, in most cases this will be done as part of a lab so the instructor will also advise.

Customizing the App

Use the Theme menu to change the appearance of the app. Your preference will be remembered next time you start Apps'Ere!.

Favourites can be re-arranged by clicking on the lower right move area on each of the icons. This layout will be remembered on next launch.
